# chon-dev-portfolio

This repository contains the source code for my personal developer portfolio website. The site serves as a digital space to showcase my journey as an ICT student, highlight my technical skills, display my web development projects, and provide a direct way for people to get in touch with me.

Check Here: https://chon-dev-portfolio.netlify.app

## About the Project
The portfolio is designed to present my background as a Senior High School IT student based in Naic, Cavite, Philippines. It illustrates my passion for front-end web development and my ongoing transition into back-end technologies. The design is kept clean, responsive, and minimalist to focus purely on content, user experience, and professional presentation.

## Key Features
- Hero Section: An introduction displaying my professional goals and a direct call to action to view my work.
- About Me: A breakdown of my dedication, work ethic, and academic focus.
- Skills and Strengths: A categorized layout showing my technical expertise and soft skills.
- Projects Section: A gallery of web development applications and projects built using core web technologies.
- Journey / Experience: A timeline tracking my academic achievements, club involvements, and metrics.
- Certificates: A dedicated space displaying verified tech credentials and online course completions.
- Interactive Contact Form: A fully functional contact section allowing users to send direct messages, paired with social links.

## Tech Stack
- HTML: Structured markup for accessibility and content layout.
- CSS: Custom styling, typography layout, and responsive media queries.
- JavaScript (ES6+): Interactive behaviors, smooth scrolling, and dynamic state handling.
- Netlify: Used for modern web hosting and continuous deployment.

## Installation and Setup
To run this project locally, follow these steps:

1. Clone the repository:
```text
Bash

git clone [https://github.com/hey-chon/chon-dev-portfolio.git](https://github.com/hey-chon/chon-dev-portfolio.git)
```
2. Navigate into the project directory:
```text
Bash

cd chon-dev-portfolio
```
3. Open the project in your preferred browser:

You can double-click the index.html file inside your file explorer.

Alternatively, if you use Visual Studio Code, use the "Live Server" extension to host it locally on localhost.
